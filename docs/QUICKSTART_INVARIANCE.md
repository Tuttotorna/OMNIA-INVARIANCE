# Quickstart Invariance

This document gives the fastest path to seeing OMNIA-INVARIANCE as a structural invariance layer.

The public mental model is:

    source structure -> controlled transformation -> invariance check -> stability / collapse

---

## Clean install

    git clone https://github.com/Tuttotorna/OMNIA-INVARIANCE.git
    cd OMNIA-INVARIANCE
    python -m pip install -e .
    pytest

---

## What to look for

After installation and tests, look for the smallest available example.

The point is not to prove universal truth.

The point is to identify:

    What source structure is transformed?
    What controlled transformation is applied?
    What property is checked?
    What remains stable?
    What collapses?
    What does the result not prove?

---

## Expected invariance behavior

A good OMNIA-INVARIANCE path should produce an invariance artifact.

It should not silently become:

    semantic judgment
    final decision
    truth certificate
    physical proof
    security finding
    cryptographic proof

---

## Public compression

    OMNIA-INVARIANCE checks what survives transformation.
    It does not infer meaning.
    It does not decide truth.
