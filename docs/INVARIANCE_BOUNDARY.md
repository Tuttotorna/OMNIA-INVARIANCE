# Invariance Boundary

OMNIA-INVARIANCE follows the ecosystem boundary:

    measurement != inference != decision

For this repository, the operational version is:

    invariance check != semantic truth != final decision

---

## Meaning

Transformation means:

    apply a controlled change to a source structure.

Invariance check means:

    inspect what remains stable or collapses.

Inference means:

    interpret the stability or collapse.

Decision means:

    act on the interpretation.

OMNIA-INVARIANCE checks structural survival.

It does not infer final meaning.

It does not decide.

---

## Allowed claims

OMNIA-INVARIANCE may claim:

- a source structure was transformed;
- a declared property persisted;
- a declared property changed;
- a declared property collapsed;
- a stable residue appeared;
- a result requires validation.

---

## Disallowed claims

OMNIA-INVARIANCE should not claim by itself:

- this is semantically true;
- this is semantically false;
- this is physically proven;
- this is universally complete;
- this is safe;
- this is insecure;
- this downstream decision is correct.

---

## Rule

The output must preserve the distinction:

    transformation
    invariance signal
    structural measurement
    interpretation
    external decision
