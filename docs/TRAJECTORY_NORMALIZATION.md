# OMNIA-INVARIANCE — Trajectory Normalization

## Core Problem

Trajectory-space may currently be influenced by:

- absolute magnitude
- volatility scale
- IRI amplitude
- perturbation intensity

rather than only by:

- structural shape
- trajectory geometry
- collapse form
- recovery dynamics

This creates a methodological problem.

Two systems may have:

- similar structural behavior
- similar trajectory shape

while appearing distant due to scale differences.

# Current Candidate Normalizations

## 1. Min-Max Normalization

Maps trajectories into:

[0, 1]

Possible effect:

- removes absolute amplitude dominance
- preserves relative structure

Risk:

- suppresses meaningful severity information

---

## 2. Z-Score Normalization

Centers trajectories around:

mean = 0
std = 1

Possible effect:

- highlights relative deviation behavior

Risk:

- unstable on short trajectories

---

## 3. Relative IRI Scaling

Normalizes IRI relative to:

local trajectory maximum

Possible effect:

- compares recovery shape
rather than absolute irreversibility

Risk:

- hides physically meaningful extremes

---

## 4. Volatility Normalization

Scales trajectories by:

trajectory variance

Possible effect:

- isolates shape under different noise levels

Risk:

- suppresses collapse intensity

---

## 5. Derivative Normalization

Focuses on:

trajectory change rates

rather than raw values.

Possible effect:

- highlights structural transitions

Risk:

- amplifies local noise
