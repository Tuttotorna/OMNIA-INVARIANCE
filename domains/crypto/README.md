# Crypto Domain

This folder contains the crypto-domain contribution to OMNIA-INVARIANCE.

Crypto is treated here as one structural domain among others.

The goal is not to build a cryptographic system.

The goal is to measure how cryptographic-like structures behave under controlled perturbation.

---

## Role in OMNIA-INVARIANCE

OMNIA-INVARIANCE compares structural stability profiles across domains.

Crypto is added as an additional domain because it provides:

- high structural rigidity
- strong sensitivity to perturbation
- clear collapse behavior
- measurable stability degradation

---

## Boundary

This folder does not define OMNIA-CRYPTO.

OMNIA-CRYPTO remains the source-domain project.

This folder only imports or adapts crypto-domain profiles for cross-domain invariance comparison.
